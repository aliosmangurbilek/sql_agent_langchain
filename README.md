# LangChain SQL Agent

A clean and efficient Flask application that converts natural language questions into SQL queries using LangChain and OpenRouter LLM. Features interactive chart generation and real-time progress tracking.

## Features
- 🤖 Natural language to SQL conversion
- 📊 Interactive chart generation with Vega-Lite
- 🔄 Real-time progress tracking via Server-Sent Events
- 🎨 Dark/Light theme support
- 📱 Responsive web interface
- 🔍 Database schema analysis with vector embeddings

See [docs/flowchart.md](docs/flowchart.md) for a high-level overview of how requests flow through the system.
